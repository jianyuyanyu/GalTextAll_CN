# 全面依法治国
`1997年9月`  
中国共产党第十五次全国代表大会提出依法治国  
`2017年10月18日`  
总书记在党的十九大报告中提出，成立中央全面依法治国领导小组  
加强对法治中国建设的统一领导。    
`2018年3月`  
中共中央印发《深化党和国家机构改革方案》，组建中央全面依法治国委员会  
中央全面依法治国委员会办公室设在司法部。  
`2022年10月16日`   
总书记在二十大报告中强调，坚持全面依法治国，推进法治中国建设。  

距依法治国提出已二十余载，与人民背道而驰，对法律公然挑衅破坏违抗的人，必将得到严惩
